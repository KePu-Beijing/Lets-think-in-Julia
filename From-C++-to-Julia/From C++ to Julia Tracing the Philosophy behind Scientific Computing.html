Dabbling in Julia – 2-D perturbed quantum oscillator

H^=−12m(d2/dx2+d2/dy2)+12mω(x2+y2)+σ(x−y)4H^=−12m(d2/dx2+d2/dy2)+12mω(x2+y2)+σ(x−y)4

Python implementation

sigma = 0.01
m = 1.
omega = 1.
x = y = np.linspace(-8,8,201)
dx = dy = x[1] - x[0]
X, Y = np.meshgrid(x, y)
V = 0.5 * m * omega * (X ** 2 + Y ** 2)
V1 = (X - Y) ** 4
H = -0.5 / m * (FinDiff(0,dx,2) + 
               FinDiff(1,dy,2)).matrix(V.shape) + 
               diags(V.reshape(-1)) + 
               sigma * diags(V1.reshape(-1))
energies_perturbed, states_perturbed = eigs(H, k=20, which='SR')

Julia implementation

N = 201         # Number of grid points in each direction
m = 1.          # Mass
ω = 1.
σ = 0.01
xx = range(-8, 8, length=N)
H̃ = -.5 / m * ∇²(xx) + V(m, ω, σ, xx)
λ̃, ϕ̃ = eigs(H̃, nev=20, which=:SR)
